This code was written and shared as part of the master's thesis by Maxine Chapman. Species occurence data and environmental predictor from CHELSA were combined to run species distribution models for three scenarios. The results were analysed and visualised.

Code is labelled numerically in the order it was run to replicate results (01, 02, 03). Species occurrence data is provided so the analysis can be replicated in full. The first 2 files provide code to download additional environmental data and global occurrences.
